# flask-blog-api
Simple RESTful api for a simple blog built in Flask; Meant as a learning instrument.

### Stack/Dependencies
* [Pipenv](https://github.com/pypa/pipenv)
* [Flask](https://flask.palletsprojects.com/en/1.1.x/)
* [PyTest](https://pytest.org/en/latest/)
